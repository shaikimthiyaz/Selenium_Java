# Selenium Dropdown Handling

##  Static Dropdowns

Static dropdowns in Selenium refer to interacting with **`<select>`** HTML elements â€” those classic dropdowns with predefined **`<option>`** values.

### 1. Select by Visible Text
```java
WebElement dropdown = driver.findElement(By.id("country"));
Select select = new Select(dropdown);
select.selectByVisibleText("India");
```



### 2. Select by Value Attribute

select.selectByValue("US");



### 3. Select by Index


select.selectByIndex(2)



### Check if Dropdown Supports Multi-Select


boolean isMultiple = select.isMultiple();


### Deselecting Options (Only for Multi-Select)


select.deselectAll();

select.deselectByVisibleText("India");

select.deselectByIndex(1);

select.deselectByValue("US");


### Get All Options


List<WebElement> options = select.getOptions();
for (WebElement option : options) {
    System.out.println(option.getText());
}

# Dynamic dropdown handling



### Techniques to Handle Dynamic Dropdowns in Selenium


### Use Explicit Waits


Dynamic dropdowns often load asynchronously.

Apply WebDriverWait with conditions like `ExpectedConditions.visibilityOfElementLocated` to wait until options appear.


### Locate Elements with XPath or CSS


Options may not have fixed IDs.

Use relative XPath (//ul/li[text()='Option']) or CSS selectors to target items dynamically.


### Iterate Through Options


Fetch all dropdown items with findElements.

Loop through them and click the one matching your desired text.


## example:


```
WebDriverWait wait = new WebDriverWait(driver, Duration.ofSeconds(10));

// Click dropdown
WebElement dropdown = driver.findElement(By.id("countryDropdown"));
dropdown.click();

// Wait for options to load
List<WebElement> options = wait.until(ExpectedConditions
    .visibilityOfAllElementsLocatedBy(By.xpath("//ul[@id='countryList']/li")));

// Select desired option
for (WebElement option : options) {
    if (option.getText().equals("India")) {
        option.click();
        break;
    }
}
```

## checkboxes

Checkboxes are HTML input elements (<input type="checkbox">) that allow users to select one or multiple options.

### Locate a Checkbox

```
WebElement checkbox = driver.findElement(By.id("subscribe"));
```

### Check if Selected

```
boolean status = checkbox.isSelected();
System.out.println("Is selected? " + status);
```

### Select a Checkbox

```
if (!checkbox.isSelected()) {
    checkbox.click();  // select only if not already selected
}
```

### unselect a checkbox

```
if (checkbox.isSelected()) {
    checkbox.click();  // unselect if already selected
}
```

### Handle multiple checkboxes 

```
List<WebElement> checkboxes = driver.findElements(By.xpath("//input[@type='checkbox']"));
for (WebElement box : checkboxes) {
    if (!box.isSelected()) {
        box.click();
    }
}
```
