### Actions class

The Actions class in Selenium (Java) is used to perform advanced **user interactions such as mouse and keyboard actions** that can’t be done with simple WebElement methods.

## What is the Actions Class?

Actions is a class in Selenium WebDriver that lets you simulate complex user gestures like:

> Mouse hover

> Click and hold

> Drag and drop

> Right-click (context click)

> Double-click

> Keyboard key presses

> Composite actions (chained actions)

It works with Action Builder Pattern, meaning you **build the action** → then execute it using **.perform()**;

## syntax of actions class

WebDriver driver = new ChromeDriver();
Actions actions = new Actions(driver);

### mouse hover

```
WebElement element = driver.findElement(By.id("menu"));
actions.moveToElement(element).build().perform();
```

### click

```
actions.click(element).perform();
```

### double click

```
actions.doubleClick(element).perform();
```

### right click

```
actions.contextClick(element).perform();
```

### drag & drop 

```
WebElement source = driver.findElement(By.id("drag"));
WebElement target = driver.findElement(By.id("drop"));

actions.dragAndDrop(source, target).perform();
```

### Click and Hold + Move

```
actions.clickAndHold(source)
       .moveToElement(target)
       .release()
       .perform();
```

## Keyboard Actions

### Press a Key

```
actions.sendKeys(Keys.ENTER).perform();
```
### Key Down (hold)/ Key Up(release)

```
actions.keyDown(Keys.CONTROL)
       .sendKeys("a")
       .keyUp(Keys.CONTROL)
       .perform();
```

### Chaining Multiple Actions

```
actions.moveToElement(element)
       .click()
       .sendKeys("Hello")
       .doubleClick()
       .perform();
```

### Keys.chord()

is used to combine multiple keys into a single key sequence. For example, Keys.chord(Keys.CONTROL, "a") represents Ctrl + A

| Method                     | Description                      |
| -------------------------- | -------------------------------- |
| `moveToElement()`          | Hover                            |
| `click()`, `doubleClick()` | Mouse clicks                     |
| `contextClick()`           | Right-click                      |
| `dragAndDrop()`            | Drag and drop                    |
| `clickAndHold()`           | Hold left button                 |
| `sendKeys()`               | Keyboard input                   |
| `keyDown()`, `keyUp()`     | Modifier keys (CTRL, ALT, SHIFT) |
| `build()`                  | Builds the action sequence       |
| `perform()`                | Executes actions                 |
