# Encapsulation

Encapsulation is an Object-Oriented Programming (OOP) principle that:

Wraps data (variables) and code (methods) together into a single unit (class) and restricts direct access to the data.

### In Java, encapsulation is achieved by:

Making variables `private`

Providing controlled access using **public getter and setter methods**

Step 2: Use public methods to access data

    private int age;

    public void setAge(int age) {
        if (age > 0) {
            this.age = age;
        }
    }

    public int getAge() {
        return age;
    }

### encapsulation vs data hiding

| Encapsulation        | Data Hiding              |
| -------------------- | ------------------------ |
| Concept              | Technique                |
| Wrap data & methods  | Restrict access          |
| Achieved using class | Achieved using `private` |

Data hiding is part of encapsulation
