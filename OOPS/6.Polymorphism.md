# Compile-time polymorphism (method overloading)

Achieved using **method overloading**.

**Same method name**, **different parameter** lists 

Which method is called is **decided at compile time**.

## Example:

```
class Calculator {

    // add two ints
    int add(int a, int b) {
        return a + b;
    }

    // add three ints (overloaded)
    int add(int a, int b, int c) {
        return a + b + c;
    }

    // add two doubles (overloaded)
    double add(double a, double b) {
        return a + b;
    }
}
```

**usage:**
```
Calculator calc = new Calculator();
System.out.println(calc.add(1, 2));        // calls add(int, int)
System.out.println(calc.add(1, 2, 3));     // calls add(int, int, int)
System.out.println(calc.add(1.5, 2.5));    // calls add(double, double)
```

# Runtime polymorphism (method overriding)

Achieved using method overriding.

A subclass provides its own implementation for a method defined in the superclass.

Which implementation runs is decided at runtime, based on the actual object.

‚úî Same method name

‚úî Same parameters

‚úî IS-A relationship (inheritance)

‚úî Access level cannot be reduced

‚úî Happens at runtime

### example:

```
class Animal {
    public void sound() {
        System.out.println("Some generic animal sound");
    }
}

class Dog extends Animal {
    @Override
    public void sound() {
        System.out.println("Dog barks");
    }
}

class Cat extends Animal {
    @Override
    public void sound() {
        System.out.println("Cat meows");
    }
}
```

**usage:**

```public class TestPolymorphism {
    public static void main(String[] args) {
        Animal a1 = new Dog();  // upcasting
        Animal a2 = new Cat();

        a1.sound(); // Dog barks
        a2.sound(); // Cat meows
    }
}
```

The reference type is Animal.

The object type is Dog or Cat.

At runtime, Java decides which sound() method to call ‚Üí runtime polymorphism.


| Feature           | Overloading  | Overriding |
| ----------------- | ------------ | ---------- |
| Polymorphism type | Compile-time | Run-time   |
| Parameters        | Must change  | Same       |
| Inheritance       | Not required | Required   |
| Binding           | Early        | Late       |
