# program to find duplicates in array

```
package practice;

public class RemoveDuplicatesInArray {

	public static void main(String[] args) {

//		using nested for loop
		
		int[] a= {3,5,4,3,6,2,7,1,2};
		
		for(int i=0;i<a.length;i++) {
			for(int j=i+1;j<a.length;j++) {
				if(a[i]==a[j]) {
					System.out.println("found duplicates are :" +a[i]);
					break;
				}
			}
		}
	}

}
```

## Explanation for logic1

We store numbers in an integer array `arr`.

The outer loop picks one element at a time.

The inner loop compares it with the remaining elements.

If two elements are equal, it means a duplicate is found.

break stops checking further to avoid repeated output.

### output: 

found duplicates are :3

found duplicates are :2


# Logic 2 using Hashset

````
Logic 2 using HashSet
		
		HashSet<Integer> set = new HashSet<>();
		System.out.println("Duplicate elemnets are:");
		
		for(int n:a) {
			if(!set.add(n)) {
				System.out.println(n);
			}
		}
```

## Explanation

HashSet stores unique values only.

set.add(num) returns:

true → if the element is new

false → if the element already exists

If add() returns false, the element is a duplicate.

This method is faster and cleaner.

### output:

Duplicate elemnets are:
3
2


### Execution :

int[] arr = {2, 4, 5, 2};


| Number (`num`) | `set.add(num)` | `!set.add(num)` | Action              |
| -------------- | -------------- | --------------- | ------------------- |
| 2              | true           | false           | Not printed         |
| 4              | true           | false           | Not printed         |
| 5              | true           | false           | Not printed         |
| 2              | false          | true            | Printed (duplicate) |
