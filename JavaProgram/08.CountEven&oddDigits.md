# counting number of even & odd digits in a number

```
package practice;

public class CountEvenOddDigits {

	public static void main(String[] args) {

		int num = 9870778;
		int even_count =0;
		int odd_count = 0;
		
		while(num>0) {
			int rem = num%10;
			if(rem%2==0) {
				even_count++;
			}
			else {
				odd_count++;
			}
			
			num = num/10;
		}
		System.out.println("no.of even numbers is " + even_count);
		System.out.println("no.of odd numbers is " + odd_count);

	}

}
````

### output:

```
no.of even numbers is 3
no.of odd numbers is 4
```
