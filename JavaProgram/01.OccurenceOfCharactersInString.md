# counting each letter in a string 

```
package practice;
import java.util.HashMap;
import java.util.Map;

public class OccurenceofCharInString {

    public static void main(String[] args) {
        // Create a HashMap to store character counts
        Map<Character, Integer> map = new HashMap<>();

        // Input string
        String name = "Rachapalle Harshitha";

        // Convert the string to a char array
        char[] letters = name.toCharArray();

        // Iterate through each character and count occurrences
        for (char ch : letters) {
            if (!map.containsKey(ch)) {
                map.put(ch, 1);            // First time we see this character
            } else {
                int count = map.get(ch);   // Get current count
                map.put(ch, count + 1);    // Increment count by 1
            }
        }

        // Print the resulting frequency map
        System.out.println(map);
    }
}

```

### output:

{ =1, a=5, c=1, e=1, h=3, H=1, i=1, l=2, p=1, R=1, r=1, s=1, t=1}
