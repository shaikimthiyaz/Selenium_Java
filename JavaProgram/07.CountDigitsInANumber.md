# count of digits

```
package practice;

public class CountDigitsInNumber {

	public static void main(String[] args) {

		int num = 540267;
		int count =0;
		while(num!=0) {
			num =num/10;
			count++;
		}
		System.out.println("number of digits is " + count);
	}

}
```

**explanation:**

| Step | number | count |
| ---- | ------ | ----- |
| 1    | 54026  | 1     |
| 2    | 5402   | 2     |
| 3    | 540    | 3     |
| 4    | 54     | 4     |
| 5    | 5      | 5     |
| 6    | 0      | 6     |

### output:

number of digits is 6
