
# finding max and min value in array

```
package practice;

import java.util.Arrays;

public class FirstMaxMinNumInArray {

	public static void main(String[] args) {

		int[] a = {46,54,32,69,9,50,13};
		
//		**Logic 1 : sorting array**

		Arrays.sort(a);
		int min = a[0];
		int max = a[a.length-1];
		
//		**Logic 2 :** using streams

		int max1 = Arrays.stream(a).max().getAsInt();
		int min1 = Arrays.stream(a).min().getAsInt();
		
//		**Logic 3:** using for and if loop
		
		int max2=a[0];
		int min2=a[0];
		for(int i=1;i<a.length;i++) {
			if(a[i]>max) {
				max=a[i];				
			}
			if(a[i]<min) {
				min=a[i];
			}
		}
	
		System.out.println("max number is " + max);
		
		System.out.println("min number is "+ min);
	}

}
```

### output:

max number is 69
min number is 9

## Logic 1 Using Arrays.sort()

Sorting arranges numbers in ascending order.

First element becomes the minimum.

Last element becomes the maximum.
