# LinkedList

LinkedList is a **linear data structure** based on nodes.

Each node stores data + links.

Java LinkedList is a **Doubly Linked List.**

Package: java.util

Implements List, Queue, Deque

## Declaration

```
LinkedList<Integer> list = new LinkedList<>();
```

Node-based storage

### Adding elements
```
list.add(10);        // end
list.addFirst(5);    // beginning
list.addLast(20);    // end
list.add(1, 15);     // index
```

### Removing elements

```
list.remove();        // first
list.removeFirst();
list.removeLast();
list.remove(1);       // index
```
