# What is LinkedHashMap?

LinkedHashMap is a class in java.util package.

It extends HashMap and implements Map interface.

Stores key-value pairs.

Maintains **insertion** order (or access order if specified).

Allows one null key and multiple null values.

Slightly slower than HashMap due to ordering overhead

```
Map<String, String> map = new LinkedHashMap<>();
```

| Feature        | HashMap         | LinkedHashMap                               |
| -------------- | --------------- | ------------------------------------------- |
| Order          | No order        | Maintains insertion order (or access order) |
| Implementation | Hash table      | Hash table + Doubly linked list             |
| Performance    | Slightly faster | Slightly slower than HashMap                |
| Null Key       | Yes (1)         | Yes (1)                                     |
| Null Values    | Yes             | Yes                                         |


## example

```
import java.util.LinkedHashMap;
import java.util.Map;

public class LinkedHashMapExample {
    public static void main(String[] args) {
        // Create LinkedHashMap
        LinkedHashMap<Integer, String> map = new LinkedHashMap<>();

        // Add key-value pairs
        map.put(1, "Apple");
        map.put(2, "Banana");
        map.put(3, "Mango");
        map.put(4, "Banana"); // duplicate value allowed
        map.put(null, "Orange"); // null key allowed

        System.out.println("LinkedHashMap: " + map);

        // Get value for a key
        System.out.println("Value for key 2: " + map.get(2));

        // Remove a key
        map.remove(1);
        System.out.println("After removing key 1: " + map);

        // Check if key exists
        System.out.println("Contains key 3? " + map.containsKey(3));

        // Iterate over map
        for (Map.Entry<Integer, String> entry : map.entrySet()) {
            System.out.println("Key: " + entry.getKey() + ", Value: " + entry.getValue());
        }
    }
}
```
**output:**
```
LinkedHashMap: {1=Apple, 2=Banana, 3=Mango, 4=Banana, null=Orange}
Value for key 2: Banana
After removing key 1: {2=Banana, 3=Mango, 4=Banana, null=Orange}
Contains key 3? true
Key: 2, Value: Banana
Key: 3, Value: Mango
Key: 4, Value: Banana
Key: null, Value: Orange
```
