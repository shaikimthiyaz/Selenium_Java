# What is HashMap?

HashMap is the most commonly used Map implementation in Java. It stores key-value pairs in a hash table.

HashMap is a class in java.util that implements the Map interface.
```
Map<String, String> map = new HashMap<>();
```

## Key Characteristics

Stores key–value pairs

No ordering guaranteed

Allows one null key

Allows multiple null values

Not thread-safe

| Feature               | Description                        |
| --------------------- | ---------------------------------- |
| Interface Implemented | `Map<K,V>`                         |
| Null Key              | Allowed (only 1)                   |
| Null Value            | Allowed (multiple)                 |
| Order                 | No guaranteed order                |
| Synchronization       | Not synchronized (not thread-safe) |
| Performance           | Fast for get() and put()           |


| Method                 | Description                 | Example                               |
| ---------------------- | --------------------------- | ------------------------------------- |
| `put(key, value)`      | Adds key-value pair         | `map.put(1, "Apple");`                |
| `get(key)`             | Returns value for key       | `map.get(1); // Apple`                |
| `remove(key)`          | Removes key-value pair      | `map.remove(2);`                      |
| `containsKey(key)`     | Checks if key exists        | `map.containsKey(3); // true`         |
| `containsValue(value)` | Checks if value exists      | `map.containsValue("Mango"); // true` |
| `size()`               | Number of entries           | `map.size();`                         |
| `keySet()`             | Returns all keys            | `map.keySet();`                       |
| `values()`             | Returns all values          | `map.values();`                       |
| `entrySet()`           | Returns all key-value pairs | `map.entrySet();`                     |


## How HashMap Works Internally

HashMap uses a combination of array + linked list (or tree in Java 8+) called **buckets**.

Step by step:

Hashing the Key

Key’s hashCode() is calculated → maps to an index in the internal array.

This index is called a bucket.

Collision Handling

Two keys can have the same bucket index → called a collision.

Java uses LinkedList or Red-Black Tree (for large buckets) to store multiple entries in one bucket.

Adding an Entry

Compute the bucket index using hash(key).

Check if key exists:

Yes → Update value

No → Add new node

Getting an Entry

Compute bucket index using key.

Traverse linked list (or tree) in that bucket to find the key.

## iteration

```
for (Integer key : map.keySet()) {
    System.out.println("Key: " + key + ", Value: " + map.get(key));
}

// Or using entrySet
for (var entry : map.entrySet()) {
    System.out.println(entry.getKey() + " = " + entry.getValue());
}
```
