# What is LinkedHashSet?

LinkedHashSet is a Set implementation that:

Does NOT allow duplicates

**Maintains insertion order**

Allows one null

Slightly slower than HashSet

```
LinkedHashSet<String> set = new LinkedHashSet<>();
```

| Feature         | LinkedHashSet                |
| --------------- | ---------------------------- |
| Duplicates      | ❌ Not allowed                |
| Insertion order | ✅ Maintained                 |
| Null values     | ✅ One null                   |
| Thread-safe     | ❌ No                         |
| Performance     | Slightly slower than HashSet |


## How LinkedHashSet Works Internally

1.**Hashing** is used to store elements (like HashSet)

2. A **doubly linked list** maintains insertion order

3. When adding an element:

hashCode() decides bucket

equals() checks duplicates

Duplicate → ignored

### example:

```
import java.util.*;

public class LinkedHashSetExample {
    public static void main(String[] args) {

        LinkedHashSet<String> set = new LinkedHashSet<>();

        set.add("Login");
        set.add("Signup");
        set.add("Login"); // duplicate
        set.add("Checkout");

        System.out.println(set);
    }
}
```
**output:**

[Login, Signup, Checkout]
