# What is a HashSet in Java?

A HashSet is a collection in Java that:

Stores **unique elements** only

Does not maintain insertion order

Is part of the java.util package

Uses a **hash table internally**


```
HashSet<String> set = new HashSet<>();
```

| Feature             | HashSet          |
| ------------------- | ---------------- |
| Duplicates allowed? | ❌ No             |
| Order maintained?   | ❌ No             |
| Null allowed?       | ✅ One null       |
| Thread-safe?        | ❌ No             |
| Performance         | Very fast (O(1)) |

## How HashSet Works Internally 

1.When you add an element:

hashCode() is called

Element is stored in a bucket

2. If another element has the same hash:

equals() is used to check duplication

3. If equals() returns true → element is not added

### example:

```
import java.util.*;

public class HashSetExample {
    public static void main(String[] args) {

        HashSet<String> users = new HashSet<>();

        users.add("Alice");
        users.add("Bob");
        users.add("Alice"); // Duplicate

        System.out.println(users);
    }
}
```

**output:**
[Alice, Bob]

| Feature      | HashSet     | ArrayList     |
| ------------ | ----------- | ------------- |
| Duplicates   | ❌ No        | ✅ Yes         |
| Order        | ❌ No        | ✅ Yes         |
| Search speed | Fast (O(1)) | Slower (O(n)) |
| Use case     | Uniqueness  | Ordered data  |

## Remove duplicates from list

```
List<Integer> list = Arrays.asList(1, 2, 2, 3, 4);
Set<Integer> set = new HashSet<>(list);
System.out.println(set);
```

## Check duplicate elements in a List

```
List<String> list = Arrays.asList("A", "B", "A");
Set<String> set = new HashSet<>();

for (String s : list) {
    if (!set.add(s)) {
        System.out.println("Duplicate found: " + s);
    }
}
```
Duplicate found: A

## find first repeating element

```
List<Integer> list = Arrays.asList(10, 20, 30, 20, 40);
Set<Integer> set = new HashSet<>();

for (int num : list) {
    if (!set.add(num)) {
        System.out.println("First repeating: " + num);
        break;
    }
}
```

First repeating: 20
