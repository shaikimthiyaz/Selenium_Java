### 1. Why HashSet allows one null?

HashSet can store null because it uses buckets, and one bucket can hold null.

No comparison is required for null.

### 2. Why TreeSet doesn’t allow null?

TreeSet sorts elements using compareTo/Comparator.

Null cannot be compared → throws NullPointerException.

## 3. Compare two lists ignoring order & duplicates

```
import java.util.*;

public class CompareLists {
    public static void main(String[] args) {

        List<Integer> list1 = Arrays.asList(10, 20, 30, 30, 40);
        List<Integer> list2 = Arrays.asList(40, 30, 20, 10, 10);

        // Convert Lists to Sets
        Set<Integer> set1 = new HashSet<>(list1);
        Set<Integer> set2 = new HashSet<>(list2);

        // Compare sets
        if (set1.equals(set2)) {
            System.out.println("Both lists are equal (ignoring order & duplicates)");
        } else {
            System.out.println("Lists are NOT equal");
        }
    }
}
```


## 4.Find union of two sets (no duplicates)

 Combine two lists removing duplicates.

```
import java.util.*;

public class UnionSet {
    public static void main(String[] args) {
        Set<String> set1 = new HashSet<>(Arrays.asList("A", "B", "C"));
        Set<String> set2 = new HashSet<>(Arrays.asList("B", "C", "D"));

        set1.addAll(set2); // union

        System.out.println("Union: " + set1);
    }
}
```

Union: [A, B, C, D]

## 5.Find intersection of two sets

Get common elements from two sets.
```
import java.util.*;

public class IntersectionSet {
    public static void main(String[] args) {
        Set<String> set1 = new HashSet<>(Arrays.asList("A", "B", "C"));
        Set<String> set2 = new HashSet<>(Arrays.asList("B", "C", "D"));

        set1.retainAll(set2); // intersection

        System.out.println("Intersection: " + set1);
    }
}
```

Intersection: [B, C]

## 7.count unique characters in a string

```
package DemoCollections;

import java.util.HashSet;

public class CountUniqueCharsInString {

	public static void main(String[] args) {

        String str = "Selenium Automation";
        HashSet<Character> chars = new HashSet<>();
        for(char c:str.toCharArray()) {
        	if(c != ' ') {
        		chars.add(c);
        		
        	}
        }
        System.out.println("Unique characters: " + chars.size());
        System.out.println("Characters: " + chars);

	}

}
```

Unique characters: 11
Characters: [A, a, S, t, e, u, i, l, m, n, o]
