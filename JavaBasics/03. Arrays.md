# Arrays

An array is a fixed-size, indexed, ordered collection of elements of the same data type

### Key Characteristics of Arrays

**Fixed size:** Once created, size cannot change

**Indexed**: Index starts at 0

**Homogeneous**: All elements must be same data type

Stored in contiguous memory: Fast access (O(1))

Can store primitives or objects

### Declaring arrays
```
int[] arr;   // recommended
int arr2[];  // also valid
```

## creating arrays

## 1.Reverse an integer array

```
package arraysPractice;

import java.util.Arrays;

public class ReverseAnIntArray {

	public static void main(String[] args) {
		int[] arr= {10,20,30,40,50,60};
		int i=0, j=arr.length-1,temp;
		while(i<j) {
			temp=arr[i];
			arr[i]=arr[j];
			arr[j]=temp;
			i++;
			j--;
		}
		System.out.println("reversed array is : " );
		for(int num:arr) {
			System.out.print(num + " ");
		}
		

	}

}
```

## 3. Linear search in array

Linear search is the simplest searching algorithm.
You check each element one by one until:

You find the target value, or

You reach the end of the array

It works on any array â€” sorted or unsorted.


```
package arraysPractice;

public class LinearSearchInArray {

	public static void main(String[] args) {

		int[] a = {67,76,87,9,51,14,62};
		int key = 14;
		boolean flag = false;
		for(int i=0;i<a.length;i++) {
			if(a[i]==key) {
				System.out.println("Element found at :" + i);
				flag=true;
				break;
			}
		}
		if(flag==false) {
			System.out.println("element not found");
		}
	}

}
```

**output:** 

Element found at : 5

## Binary search

array order should alwaysbe in sorted

```
package arraysPractice;

public class BinarySearchInSortedArray {

	public static void main(String[] args) {

		int[] a= {2,3,6,9,10,15,30,35,45};
		boolean flag = false;
		int key =9;
		int l=0;
		int h=a.length-1;
		while(l<=h) {
			int m= (l+h)/2;
			if(a[m]==key) {
				System.out.println("element found at index: "+ m);
				flag=true;
				break;
			}
			else if(a[m]<key) {
				l=m+1;
			}
			else {
			h=m-1;
			}
		}
		if(flag == false) {
			System.out.println("element not found");
		}
	}

}
```

**output:**

element found at index: 3
