
# finding the min and max value in multi dimensional array 

```
package arraysPractice;

public class minAndMaxNumberInMultiDimensionalArray {

	public static void main(String[] args) {

		int[][] a= {{5,3,2},{12,7,3},{4,1,9}};
		
		int min =a[0][0];
		int max =a[0][0];
		for(int i=0;i<3;i++) { //row
			for(int j=0;j<3;j++) {
				if(a[i][j]<min) {
					min=a[i][j];
				}
				else if(a[i][j]>max) {
					max= a[i][j];
				}
				
			}
		}
		System.out.println("min value is " + min );
		System.out.println("max value is " + max );

		
	}

}
```

### output:

min value is 1
max value is 12
