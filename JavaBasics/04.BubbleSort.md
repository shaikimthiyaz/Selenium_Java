# Bubble sorting

```
package arraysPractice;

import java.util.Arrays;

public class BubbleSort {

	public static void main(String[] args) {

		int[] a= {8,7,4,6,0,3,5};
		int n= a.length-1;  //7-1=6
		int temp =0;
		
		for(int i=0;i<n;i++) {
			for(int j=0;j<n;j++) {
				if(a[j]>a[j+1]) {
					temp = a[j];
					a[j]=a[j+1];
					a[j+1]=temp;
					
				}
			}
		}
	
		System.out.println("after sort is : "+ Arrays.toString(a));
	}

}
```

**output** : [0,3,5,6,7,8]
